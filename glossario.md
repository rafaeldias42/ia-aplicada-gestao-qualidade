# Glossário

| Termo | Definição |
|---|---|
| Inteligência Artificial | Campo que desenvolve sistemas capazes de executar tarefas associadas à inteligência humana. |
| IA Generativa | Tipo de IA capaz de gerar textos, imagens, resumos, códigos e outros conteúdos. |
| Machine Learning | Técnica em que modelos aprendem padrões a partir de dados. |
| Deep Learning | Subárea de machine learning baseada em redes neurais profundas. |
| Prompt | Instrução fornecida a um sistema de IA para orientar sua resposta. |
| Engenharia de Prompts | Prática de estruturar comandos para obter respostas melhores de sistemas de IA. |
| Alucinação | Resposta incorreta ou inventada produzida por IA com aparência de confiança. |
| NotebookLM | Ferramenta de IA voltada ao estudo e exploração de fontes selecionadas. |
| Governança de IA | Conjunto de políticas, responsabilidades e controles para uso seguro e responsável de IA. |
| Viés Algorítmico | Tendência de um modelo produzir resultados distorcidos por dados ou regras enviesadas. |
| Gestão da Qualidade | Práticas usadas para garantir atendimento a requisitos e melhoria contínua. |
| SGQ | Sistema de Gestão da Qualidade. |
| ISO 9001 | Norma internacional de requisitos para sistemas de gestão da qualidade. |
| Qualidade 4.0 | Integração entre qualidade, dados, automação, conectividade e tecnologias digitais. |
| PDCA | Ciclo Plan, Do, Check, Act para melhoria contínua. |
| DMAIC | Método Define, Measure, Analyze, Improve e Control usado em Six Sigma. |
| Kaizen | Filosofia de melhoria contínua baseada em pequenas melhorias frequentes. |
| Lean Manufacturing | Abordagem para eliminar desperdícios e aumentar valor ao cliente. |
| CEP | Controle Estatístico de Processo, usado para monitorar variações. |
| OEE | Indicador de eficiência global de equipamentos. |
| CAPA | Ações corretivas e preventivas para tratar causas de problemas. |
| Não Conformidade | Não atendimento a um requisito especificado. |
| Ação Corretiva | Ação para eliminar a causa de uma não conformidade e evitar recorrência. |
| Correção | Ação imediata para eliminar uma não conformidade detectada. |
| Causa Raiz | Causa fundamental de um problema. |
| 5 Porquês | Técnica de questionamento sucessivo para investigar causa raiz. |
| Ishikawa | Diagrama de causa e efeito usado para organizar hipóteses de causa. |
| Auditoria Interna | Avaliação sistemática para verificar conformidade e eficácia de processos. |
| Evidência Objetiva | Dado, registro ou fato verificável que sustenta uma conclusão. |
| Indicador de Desempenho | Medida usada para acompanhar resultados de processo ou organização. |
| Gestão Documental | Controle de documentos, versões, aprovações e distribuição. |
| Rastreabilidade | Capacidade de acompanhar histórico, aplicação ou localização de um item ou decisão. |
| Melhoria Contínua | Busca recorrente por aumento de desempenho e redução de problemas. |
| Gestão do Conhecimento | Prática de capturar, organizar, compartilhar e aplicar conhecimento organizacional. |
| Análise Preditiva | Uso de dados para estimar comportamentos ou resultados futuros. |
| Dados Mestres | Dados essenciais e padronizados usados por processos organizacionais. |
| Data Quality | Qualidade dos dados quanto a precisão, completude, consistência e atualidade. |
| Validação Humana | Revisão feita por pessoa qualificada antes de aceitar uma saída da IA. |
| Segurança da Informação | Proteção de dados contra acesso, uso, alteração ou divulgação indevida. |
| Risco | Efeito da incerteza sobre objetivos. |
